# VisualInterpreter

VisualInterpreter é um projeto desenvolvido para a análise e manipulação visual de dados utilizando Python e ferramentas como OpenCV e NumPy. <br> Ele oferece uma interface gráfica interativa para realizar operações visuais e interpretar os resultados de forma prática, <br> com suporte para funcionalidades como captura de vídeo e aplicação de filtros.

## Descrição do Projeto

O objetivo do VisualInterpreter é criar uma aplicação simples que demonstre como usar bibliotecas populares como OpenCV para criar interfaces interativas que processam e exibem dados visuais. <br> Ele pode ser utilizado como base para projetos mais avançados em visão computacional ou apenas para aprendizado.

## Funcionalidades

1. Captura de vídeo em tempo real usando a câmera do dispositivo.
2. Aplicação de filtros visuais, como conversão para escala de cinza.
3. Interface com botões interativos para controle das funcionalidades.
4. Suporte para fechamento seguro do aplicativo e liberação dos recursos de câmera.

## Estrutura do Projeto

O repositório possui os seguintes elementos principais:
  Script principal: Contém o código base para inicializar a interface e gerenciar a lógica dos eventos.
  Funções de manipulação visual:
    Captura e exibição do feed da câmera.
    Aplicação de transformações visuais (ex.: escala de cinza).
  Gestão de eventos de clique para interação com os botões.
  Recursos reutilizáveis para desenhar elementos visuais, como botões.

## Tecnologias Utilizadas

Python 3.8+
OpenCV: Manipulação e processamento de imagens e vídeos.
NumPy: Suporte matemático para operações de matriz.
Interface Gráfica: Desenvolvida diretamente com os recursos do OpenCV.
